# Как открыть Сапёр в Telegram как мини-приложение

Игра уже подготовлена для Telegram (поддержка темы, полноэкранный режим, долгое нажатие для флага на телефоне). **Aiogram не нужен** — мини-приложение это просто страница по HTTPS.

---

## Что нужно сделать

### 1. Создать бота в Telegram (если ещё нет)

1. Откройте Telegram и найдите **@BotFather**.
2. Отправьте команду `/newbot`.
3. Введите имя бота (например: `Сапёр`) и username (например: `MyMinesweeperBot`).
4. BotFather пришлёт вам **токен** вида `123456789:AAH...` — сохраните его (для мини-приложения он не обязателен, но пригодится для кнопки меню).

---

### 2. Разместить игру в интернете по HTTPS

Telegram открывает мини-приложения только по **HTTPS**. Варианты бесплатного хостинга:

#### Вариант A: GitHub Pages (простой)

1. Зарегистрируйтесь на [github.com](https://github.com).
2. Создайте новый репозиторий (например `minesweeper-tg`).
3. Загрузите в репозиторий **один файл** — `index.html` из папки `minesweeper`.
4. В настройках репозитория: **Settings → Pages → Source**: выберите ветку `main` и папку `/ (root)` → Save.
5. Через пару минут сайт будет доступен по адресу:  
   `https://ВАШ_ЛОГИН.github.io/minesweeper-tg/`  
   (подставьте свой логин и имя репозитория).

#### Вариант B: Vercel (без регистрации на GitHub)

1. Зайдите на [vercel.com](https://vercel.com) и войдите через GitHub.
2. **Add New → Project** → импортируйте репозиторий с `index.html` или загрузите папку через **Vercel CLI**.
3. Либо: установите [Vercel CLI](https://vercel.com/cli), в папке `minesweeper` выполните `vercel` и следуйте подсказкам.
4. Vercel выдаст ссылку вида `https://ваш-проект.vercel.app`.

#### Вариант C: Netlify

1. Зайдите на [netlify.com](https://netlify.com).
2. Перетащите папку `minesweeper` (с `index.html` внутри) в зону **Drag and drop your site**.
3. Сайт появится по адресу вида `https://случайное-имя.netlify.app` (имя можно поменять в настройках).

---

### 3. Привязать мини-приложение к боту

После того как у вас есть **HTTPS-ссылка** на игру (например `https://username.github.io/minesweeper-tg/`):

**Способ 1 — кнопка меню (рекомендуется)**

1. Напишите боту @BotFather команду `/mybots`.
2. Выберите своего бота → **Bot Settings** → **Menu Button** → **Configure menu button**.
3. Укажите:
   - **URL:** ваша ссылка на игру (например `https://username.github.io/minesweeper-tg/`);
   - **Button text:** например `Играть в Сапёр`.
4. Готово. Когда пользователь откроет бота, слева внизу появится кнопка меню — по нажатию откроется ваше мини-приложение.

**Способ 2 — команда или описание**

- В описании бота (BotFather → Edit Bot → Edit Description) можно написать: «Играть: [ваша ссылка]».
- Или создать команду типа `/play`, которая в ответ присылает сообщение с кнопкой (Inline Keyboard), ведущей на вашу ссылку. Для этого уже нужен сервер и библиотека (aiogram или другой бот-фреймворк). Для начала достаточно кнопки меню из способа 1.

---

## Кратко

| Шаг | Действие |
|-----|----------|
| 1 | Создать бота в @BotFather |
| 2 | Выложить `index.html` на GitHub Pages / Vercel / Netlify и получить HTTPS-ссылку |
| 3 | В @BotFather: **Bot Settings → Menu Button** → вставить ссылку и текст кнопки |

После этого при открытии бота в Telegram будет кнопка, по нажатию откроется Сапёр как мини-приложение. В браузере тот же `index.html` по-прежнему работает как обычная страница (и через localhost, и по HTTPS).
