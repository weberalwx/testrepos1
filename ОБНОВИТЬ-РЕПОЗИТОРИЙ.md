# Команды для обновления репозитория на GitHub

После изменений (видео после победы и т.д.) выполните в папке игры:

```powershell
cd C:\Users\gusev\Desktop\minesweeper

git add .
git status
git commit -m "Видео после победы, окошко чуть больше поля"
git push
```

Если репозиторий ещё не привязан:

```powershell
cd C:\Users\gusev\Desktop\minesweeper
git remote add origin https://github.com/weberalwx/testrepos1.git
git add .
git commit -m "Видео после победы"
git branch -M main
git push -u origin main
```

После `git push` на GitHub будут обновлены **index.html** и добавлен файл **victory.mp4** (видео из папки videoASU).
